// CONTEXT: PROBLEM & OBJECTIVE

The Barangay Fix was conceptualized to address the heavy reliance on manual, in-person processes at local government offices. This often led to long wait times, lost paperwork, and inefficiency in addressing community needs.

// SOLUTION EXECUTION

The solution was a comprehensive web portal built directly on the **ServiceNow** platform. This leveraged ServiceNow's powerful workflow engine and security framework. The frontend was custom-built using **AngularJS** for dynamic functionality and **Bootstrap** for a mobile-responsive and familiar user interface.

KEY FEATURES

  • Resident Issue Reporting (Geo-location aware).
  • Automated ticket routing to relevant departments.
  • Digital document request and issuance tracking.
  • Official dashboard for managing ticket SLAs.
