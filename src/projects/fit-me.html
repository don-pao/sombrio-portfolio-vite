// CONTEXT: MOBILE DESIGN CHALLENGE

FIT Me was a school capstone project aimed at building a robust, full-stack mobile application that handled user authentication, workout logging, and progress visualization. The primary goal was to demonstrate proficiency in native mobile development practices and backend API design.

// SOLUTION EXECUTION

The application utilized **Kotlin** for the native Android frontend, ensuring high performance and access to device features like the **native Calendar** for scheduling workouts. The core business logic and database connectivity were handled by a separate **Java**-based RESTful API server. **Google Login** was integrated for fast, secure user onboarding.

KEY FEATURES

  • Secure **Google OAuth** for quick sign-in.
  • Seamless integration with native phone calendars.
  • Dynamic data visualization of progress and goals.
**[PROJECT STATUS NOTE]** This project was fully functional but was discontinued after the school term due to the ongoing **server and deployment fees** required to maintain the Java backend API.
